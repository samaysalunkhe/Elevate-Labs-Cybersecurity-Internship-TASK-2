# 🎣 Task 2 - Phishing Email Analysis

## 📌 Objective

Analyze a suspicious email sample and identify phishing indicators to understand common phishing techniques.

## 🛠️ Tools Used

* 📄 Sample Phishing Email
* 🌐 Free Online Email Header Analyzer *(Optional)*
* 💻 VS Code / Notepad

## 🔍 Phishing Indicators Identified

* 🚩 Fake sender email address
* 🔗 Suspicious URL
* ⚠️ Urgent and threatening language
* 👤 Generic greeting
* 🎭 Possible email spoofing
* 🔒 Credential theft attempt

## ✅ Conclusion

This task helped identify common phishing characteristics such as spoofed sender addresses, suspicious links, and social engineering techniques. Understanding these indicators improves the ability to recognize and avoid phishing attacks.
